#  Cruise Cinema

Olá professor e visitantes! Bem-vindos ao repositório do **Cruise Cinema**. Nosso site foi desenvolvido com muito esforço e tempo dedicado, como nosso projeto prático de Avaliação Final para a disciplina de Linguagens de Programação.

## O Tema escolhido
Nós pensamos em fugir um pouco do comum, então escolhemos focar na carreira, nos grandes sucessos e nas sagas icônicas do ator e produtor **Tom Cruise**.

## O que você vai encontrar por aqui (Páginas do Projeto)
Criamos um ecossistema completo com 4 páginas principais:
- **Home (`index.html`):** Nossa porta de entrada, com direito a um carrossel dinâmico e atalhos diretos para as grandes franquias.
- **Galeria (`galeria.html`):** Um catálogo imersivo. Criamos cards expansíveis para que o usuário leia a sinopse sem sair da página.
- **Contato (`contato.html`):** Uma área de comunicação blindada contra mensagens vazias, junto com o mapa interativo do nosso "QG".
- **Biografia (`biografia.html`):** Uma página extra dedicada a contar a trajetória de Hollywood do astro.

## O que foi usado no projeto
- **HTML5:** Com tags semânticas, mantendo o código limpo e organizado.
- **CSS3:** Todo o visual dark/gold foi feito por nós. Usamos muito Flexbox para deixar o layout no lugar certo.
- **Bootstrap 5:** Nosso melhor amigo para agilizar a Navbar responsiva, o sistema de Grid, o Carrossel e o Modal.
- **JavaScript:** Usado estrategicamente para trazer interatividade real e manipular o DOM.
- **Git & GitHub:** Para o versionamento, salvando nosso progresso a cada etapa.

## Funcionalidades 
- **Menu Responsivo:** Adapta perfeitamente para quem acessar pelo celular.
- **Carrossel Automático:** Na página inicial, passando os maiores sucessos do ator.
- **Formulário Inteligente:** Validação via JS que trava o envio vazio e solta um Modal (Pop-up).
- **Botão "Voltar ao Topo":** Ele fica escondido e só aparece quando rola a tela para baixo.
- **Scroll Suave:** Um clique nas Sagas da Home faz a tela deslizar até os filmes escolhidos.
- **Interações Visuais:** Efeitos de hover e escala nas capas dos filmes.
